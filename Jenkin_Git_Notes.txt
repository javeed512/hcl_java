Jenkins:
---------
1. Goto Jenkins folder and open command prompt in that folder
2. Run jenkins.war file using java -jar command
 -> java -jar  jenkins.war
3. Goto any browser and open jenkin dashboard
ex: http://localhost:8181/    ---> jenkins dashboard
